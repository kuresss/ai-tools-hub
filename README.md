# AI Tools Hub

Free AI-powered online tools that run entirely in the browser. No API keys, no sign-up, 100% private.

## Tools

- **AI Text Summarizer** - Summarize long texts using Transformers.js
- **AI Sentiment Analyzer** - Detect positive/negative sentiment
- **AI Translator** - Translate between English and French/German/Spanish/Chinese
- **Word & Character Counter** - Count words, characters, sentences
- **Text Case Converter** - Convert between UPPERCASE, camelCase, snake_case, etc.

## Tech Stack

- [Astro](https://astro.build/) - Static site generator
- [Tailwind CSS v4](https://tailwindcss.com/) - Styling
- [Transformers.js](https://huggingface.co/docs/transformers.js/) - Client-side AI inference
- [Netlify](https://netlify.com/) - Free hosting

## Deploy

[![Deploy to Netlify](https://www.netlify.com/img/deploy/button.svg)](https://app.netlify.com/start/deploy?repository=https://github.com/YOUR_USERNAME/ai-tools-hub)

```bash
npm install
npm run build
# Upload dist/ to Netlify, or connect your GitHub repo
```

## Development

```bash
npm run dev    # Start dev server
npm run build  # Build for production
npm run preview # Preview production build
```
